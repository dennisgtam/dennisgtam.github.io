Technical Blog 2: Element Display Types!

July 19th, 2015

Today I'm writing about a topic that caused me a lot of grief while I was working through the challenges. I didn't even know the difference between display types. Certain positioning attributes don't work for every display type and it took me awhile to try and figure it out

Every element in a HTML file can be assigned a display type. Usually we do this inside of a css file. There are 3 primary types of displays. We'll use some colored div blocks to show what they look like.
